# CareRadar

## Zone: forge
- 나를 성장시키는 도구.
- 학습 기록(Insight) 반드시 남길 것.
- 완벽보다 꾸준함.

---

## 개요
케어테크 사업 리더를 위한 BI + 역량 개발 플랫폼 (개인용).

## 스택
React + TypeScript + Vite + Supabase + Claude API + Recharts + Tailwind

## 구조
- Trend Tracking: News Feed, Opportunity Score, Procurement Tracker
- Study: 5단계 루프 (Question→My Answer→AI Reference→Gap Check→Insight)
- Knowledge Base: 인사이트 타임라인 + 역량 대시보드

## 데이터
- 실제: DART OpenAPI, SEC EDGAR, 나라장터
- 가상: AI 생성 (라벨링 필수)
